# Phase 2 — CV Evaluation Pipeline (Not Yet Implemented)

This directory is reserved for Phase 2 functionality:

- Select job opening to evaluate against
- Define evaluation criteria
- Upload CVs (PDF/DOCX)
- AI-powered CV scoring and ranking
- Results dashboard

## Planned Architecture

1. **Job Selection** — Pick from active job openings (Phase 1 data)
2. **Criteria Configuration** — Define must-have vs nice-to-have requirements
3. **CV Upload** — Batch upload of candidate CVs
4. **AI Evaluation** — Local Ollama model scores each CV against criteria
5. **Results** — Ranked list with scores and reasoning
