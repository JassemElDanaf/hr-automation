SMTP Setup Guide

      
    

    

      

        Emails are queued into the database regardless of SMTP config. To actually deliver them, configure an SMTP credential in n8n once:
      

      
    
        
  1. Open n8n Credentials
    2. 
        
  2. Click Add credential → search SMTP → select it
    3. 
        
  3. Name it exactly SMTP
    4. 
        
  4. Fill in:
          
      
            
    • User: your-email@diyar.com
      • 
            
    • Password: app password (not your account password)
      • 
            
    • Host: smtp.gmail.com (or your provider)
      • 
            
    • Port: 587 (TLS) or 465 (SSL)
      • 
          
    
        
    5. 
        
  5. Click Save
    6. 
        
  6. Optionally set SMTP_FROM env var for the From address
    7. 
      


      

        Gmail users: you must create an App Password (requires 2FA enabled). Your normal Google password won't work.
      


      

        Testing locally? Run Mailpit (docker run -d -p 1025:1025 -p 8025:8025 axllent/mailpit) and use host localhost port 1025 with no auth. View caught mail at localhost:8025.
      


      

        How to verify it's working: send a test rejection from the CV Evaluation page. If the Email History status shows sent, you're all set. If it shows failed, hover the row to see the error.
